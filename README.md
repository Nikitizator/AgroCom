Documentation
    
    Агроком - это платформа для расчетов и прогнозирования затрат 
    на вспомогательные ресурсы в сельскохозяйственной отрасли

Features

    Отсутствие зависимостей: Работает без интернета (все ресурсы локальные).

    Локальное сохранение: Данные не пропадают при перезагрузке страницы.

    Динамический конструктор: Добавление и удаление ресурсов.

    Умная конвертация: Автоматический перевод килограмм в тонны.

    Печать и сохранение: Специальный CSS-слой для формирования 
    строгой печатной ведомости.

Stack
    
    HTML5/CSS3: Основная верстка и стилизация.

    JavaScript: работа с Web Storage API.

    CSS Media Queries: настройка печатных форм.

    Node.js: исполнение кода JavaScript

Structure
    
    index.html - верстка.

    style.css - стилизация.

    config.js - дефолтная конфигурация.

    script.js - основная логика.
